###Instruction for use
```

2.Run the main method fo the `Main` class.

*`encryption` is your game record use aes encryption.
*`hash` is your game record use block hash.
*`aesEncryptionHash` is the hash where aes encryption is. 
*`index` is the index where aes encryption is.
```
